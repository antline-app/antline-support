# Antline Support

Welcome to the official support repository for **Antline**, the modern, keyboard-friendly database console for developers.

Here you can:

- Submit **bug reports**
- Ask for **support or help**
- Suggest **new features**

Please check existing issues before creating a new one.

---

## How to Submit Issues

We use **GitHub Issues** for support and tracking. To make it easier for us to help you, please use the templates provided:

- **Bug Report** – Report a problem with Antline.
- **Support Request** – Ask a question or request help.
- **Feature Request** – Suggest a new feature or improvement.
